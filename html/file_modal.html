Custom upload of files

Save selections for reuse

After running a query, you can save a file containing a list of all stocks you have used. To download the file:

1. Click on the "sheet"-icon in the center of the footer. The file should download automatically.
2. Should your web-browser not download the file, save the file by pressing right click --> "Save".
3. Resupply the file for your next query under "Custom Upload".

Creating a CSV file

To create the CSV-file, open a text-editor or spreadsheet software. Then write two comma-separated values per line, in the form project,stock.For example:

Project Stock Explanation
NAM B73xB97 Select B73xB97 from NAM
Ames282 B73 Select B73 from Ames282
R&D All Selects all taxa in project R&D
All 05-397 Selects any matches with name 05-397

You can select all stocks from the project by writing "All" in the stock column. If you only know the stock name but not the project it belongs to, you can write 'All" in the project column to select all projects.

Creating a custom stock file

You can create a custom file by copying the stocks directly from the provided source files. On the query selection page:
1. Click on "View Sources"
2. Select the data set you would like to run your query against.
3. Click on the file you would like to open. This should open up a separate tab.
4. Simply copy & paste desired stocks into your custom file

Important: For security reasons the submission will only accept a file of type '.stockinfo', 'txt', or '.csv'.
